Detailed Description

The master controller class, that handles taking in multiple CSVs, turning each into its own Document. Creates a master list of Rooms, finds all Courses, filters Courses to relevant ones, schedules those Courses, and creates an easy access list. Also contains lookup methods for Courses and Rooms.
